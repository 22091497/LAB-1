Glossary

HTML
HyperText Markup Language, the standard language for creating web pages.
Semantic HTML
HTML that clearly describes its meaning to both the browser and the developer, using elements like <nav>, <main>, <footer>.
Favicon
The small icon displayed in a browser tab beside the page title.
Commit
A snapshot of changes in version control systems like Git, usually saved with a descriptive message.
Navigation
The system of links and menus that help users move between different pages and sections of a website.
